# A console version of the P2PSP entities #

## Pre-requisites

### Linux 
```
$ sudo apt-get install cmake
$ sudo apt-get install libboost-all-dev
```

## Cloning
```
git clone git@github.com:P2PSP/core.git
git clone git@github.com:P2PSP/p2psp-console.git
```

## Build from Source

```
cd core
./make.py
cd ../p2psp-console
./make.py
```
